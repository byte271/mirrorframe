Ledgerloom changelog

Release notes for the fictional Ledgerloom bookkeeping tool. Built as a Mirrorframe pseudo-element fixture: badges, timeline dots, underline accents, and decorative marks are all ::before/::after content.

March releases

Multi-currency journals

Journals can now hold entries in any ISO currency, with rates pinned at posting time so historical reports never drift.

CORE

Locked-period warnings

Posting into a closed period now warns before it books, instead of silently landing in the adjustments bucket.

SAFETY

CSV import mapping presets

Column mappings can be saved per bank, so the fourth import of the month takes seconds instead of minutes.

IMPORT
Ledgerloom is the only bookkeeping tool whose changelog I read for fun. The locked-period warning alone has saved our quarter-close twice.
